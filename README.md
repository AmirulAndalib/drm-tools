# drm-tools
Tools for Widevine and PlayReady

## Installation

```bash
$ npm i -g drm-tools
```

## Usage

```bash
$ drmtools
Usage: drmtools <command> -h for help

Widevine DRM Command Line Tools

Options:
  -v, --version                   output the version number
  -h, --help                      output usage information

Commands:
  widevine [options] [operation]
```
